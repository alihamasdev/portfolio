# 🚀 Ali Hamas - Personal Portfolio

Welcome to my personal portfolio! This project showcases my journey as a full-stack developer, featuring a collection of my projects, skills, and experiences.

Built with performance and aesthetics in mind, this portfolio leverages the latest web technologies to provide a seamless user experience.

---

## 🛠️ Tech Stack

This project is built using a modern and powerful stack:

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Forms & Validation**: [React Hook Form](https://react-hook-form.com/), [Zod](https://zod.dev/)
- **Runtime & Package Manager**: [Bun](https://bun.sh/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)

---

## ✨ Key Features

- **Project Showcase**: A detailed list of my web applications with tech stack badges and live links.
- **Dynamic Animations**: Smooth transitions and interactive elements using Motion.
- **Dark Mode Support**: Seamless theme switching using `next-themes`.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **MDX Support**: Integrated blog/content capabilities using `@next/mdx`.
- **SEO Optimized**: Built-in SEO best practices for better search visibility.

## 🚀 Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites

You need to have **Bun** installed on your machine. If you don't have it, install it from [bun.sh](https://bun.sh/).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/alihamasdev/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   bun install
   ```

3. **Run the development server:**
   ```bash
   bun dev
   ```

4. **Open in browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result.
